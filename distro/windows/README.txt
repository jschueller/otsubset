# otsubset module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otsubset\examples\otsubset

Installation
  Silent install can be done with the switch /S (e.g. openturns-otsubset-0.0-py27.exe /S).
  To test otsubset module installation, in an OpenTURNS python console type:
    import otsubset
    print otsubset.__version__

Uninstallation
  OTSubset module will be automatically uninstalled at OpenTURNS removal.
  OTSubset module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
